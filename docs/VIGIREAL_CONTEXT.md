# VigiReal Product Context

VigiReal is a real-time biohazard mapping app for public-health situational awareness.

## Core Hazards

- Disease outbreaks using CDC NNDSS-compatible surveillance as the priority dataset.
- Contaminated drinking or recreational water.
- Harmful algal bloom probability and field observations.
- Tick-borne illness risk using vector, habitat, climate, and case signals.
- Non-communicable disease burden as a vulnerability layer.
- Allergy and pollen signals as a seasonal burden layer.

## Product Rules

- The map is for triage and verification, not official diagnosis.
- Agency advisories remain the source of truth.
- NCD and allergy datasets should modify response planning; they should not be treated as outbreak proof.
- Crowdsourced reports should increase review priority only after quality checks.

## Missing Feature Direction

- Add CDC NNDSS connector and state-level notifiable disease reconciliation.
- Add water quality and advisory connectors.
- Add vector surveillance and habitat layers for tick-borne illness.
- Add NCD, allergy, pollen, air quality, and vulnerability context layers.
- Add role-specific workflows for residents, clinicians, and health authorities.
